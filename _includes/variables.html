Variables

We are using scss variables to manage common styles like colors, font-families and sizes, and layout width. If you find yourself needing to change the value of a variable, please see Aly first. These are located in _variables.scss.

Usage

{% highlight scss %} .class { color: $white; background-color: $blue; width: $content-width; } {% endhighlight %}

Color variables

{% highlight scss %} $white: #fff; $black: #333; $blue: #02BCDD; $red: #e74c3c; $yellow: #f1c40f; $green: #2ecc71; $purple: #8e44ad; $orange: #f39c12; {% endhighlight %}

Font variables

{% highlight scss %} $base-font-family: Helvetica, Arial, sans-serif; $base-font-size: 16px; $base-line-height: 1.5; {% endhighlight %}

Layout variables

{% highlight scss %} $content-width: 980px; {% endhighlight %}
